 <h1>🍿 usePopcorn</h1>
  <p>
    A sleek, interactive movie tracker built with <strong>React</strong>. Browse, search, and track movies you've watched — complete with ratings, posters, and runtime summaries.
  </p>

  <img src="https://github.com/user-attachments/assets/9c9806bf-824e-4071-be2e-796896d3bc73" alt="Screenshot of search results" width="100%" style="margin-bottom: 1rem;"/>

  <img src="https://github.com/user-attachments/assets/7a996011-d76e-4a1d-9ca5-f3c7ef34e765" alt="Screenshot of watched summary" width="100%" />

  <hr />

  <h2>✨ Features</h2>
  <ul>
    <li>🎬 View a curated list of movies</li>
    <li>🔎 Search movies by title</li>
    <li>⭐ Rate and track movies you’ve watched</li>
    <li>📊 View average IMDb scores, user ratings, and runtimes</li>
    <li>🎛 Toggle visibility of movie and watched lists</li>
    <li>⚡ Fast, responsive UI with clean component architecture</li>
  </ul>

  <hr />

  <h2>🧰 Tech Stack</h2>
  <ul>
    <li><strong>React</strong> (Functional Components + Hooks)</li>
    <li><strong>CSS</strong> (no external UI frameworks)</li>
    <li><strong>OMDb API</strong> (to be integrated)</li>
    <li>Local state management with React</li>
  </ul>

  <hr />

  <h2>🚀 Getting Started</h2>
  <pre><code>git clone https://github.com/your-username/use-popcorn.git
cd use-popcorn
npm install
npm start</code></pre>
  <p>
    🔑 You'll need a free <a href="http://www.omdbapi.com/apikey.aspx" target="_blank" rel="noopener noreferrer">OMDb API key</a>. Add it in <code>App.js</code> or configure it via <code>.env</code>.
  </p>

  <hr />

  <h2>📂 Project Structure</h2>
  <pre><code>src/
├── components/        # Reusable UI components (StarRating, etc.)
├── App.js             # Core app logic and layout
├── index.js           # Entry point
└── styles/            # Styling (if modularized)</code></pre>
